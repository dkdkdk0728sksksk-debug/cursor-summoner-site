Cursor Summoner

カーソルを召喚する

見失ったカーソルを、魔法陣で見つける macOS メニューバーアプリ。

Cursor Summoner は、ホットキーまたはマウスシェイクで現在のカーソル位置に Magic Circle を表示する macOS メニューバーアプリです。

大きな画面、複数ディスプレイ、細かい作業中など、カーソルの位置が一瞬わからなくなった時に、楽しく素早く見つけられます。

Main features

  • Magic Circle visual effect
  • Hotkey trigger
  • Mouse shake trigger
  • Optional Pointer Recall
  • Effect size settings
  • Reduce Motion support
  • Launch at Login support
