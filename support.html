Cursor Summoner Support

Help and contact information

If you need help with Cursor Summoner, please contact:

support@cursorsummoner.com

Before contacting support, please check the following:

1. App is running

Cursor Summoner runs as a macOS menu bar app. Please confirm that the app icon appears in the menu bar.

2. Hotkey

Open Settings and confirm that Hotkey is enabled. Press the configured hotkey to show the Magic Circle effect at the current cursor position.

3. Mouse Shake

Open Settings and confirm that Mouse Shake is enabled. Shake the mouse to show the Magic Circle effect.

4. Pointer Recall

Pointer Recall is optional and off by default. When enabled, pressing the hotkey moves the cursor to the center of the current display.

Mouse shake does not move the cursor.

5. Permissions

Cursor Summoner does not require:

  • Accessibility permission
  • Input Monitoring permission
  • Screen Recording permission

6. Local features

Cursor Summoner does not require network access for its current local visual effect features.
